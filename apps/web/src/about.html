About Multwiple

Multwiple is one of the best and easy to use web-based twitter client. The first ever public release was launched on 30th Jan, 2010. Started as a simple service to manage multiple twitter accounts, Mutlwiple has ever since become more feature rich and has been acquiring more user base.

Multwiple is an initiative of Spundhan, a fast growing startup based in India, backed by highly motivated folks ready to conquer the world!

 
Who uses Multwiple?

Multwiple is useful for everyone who wants to have a better and easy interface to twitter service.

If you have more than one twitter acounts an want to tweet using your different accounts without the much hassle? then Multwiple is for you!

If you have tweet a lot? or keep track of a keywords in tweets? or want to keep statistics? then Multwiple is for you!

 

How is Multwiple different from others?

Multwiple has been designed from ground up to be simple and easy to use. Our developers have concentrated more on providing a better user experience than just creating a glossy looks. There are several features unique to multwiple.

Take your tweet with the Link:

Often we open several links (found in tweets) in quick succession into new browser tabs to read them one after the other. The tweets contain shortened links. However, when opened the browser shows long URL (the original link).

How do we guess which link came from which tweet?
What if I want to re-tweet that tweet?

Not to worry! Multwiple marrys the tweet to the link it contains.

i.e. whenever you click on a link in a tweet, a new tab automatically opens with the top portion of the brower showing the original tweet. This way you can relate to he tweet and re-tweet it on the spot!

Cool Search:

It allows you to search for keywords in tweets around the world in real-time. To add to this, you can even search for the tweet tweeted by a particular client!
